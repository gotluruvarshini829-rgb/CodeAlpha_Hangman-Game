
Hangman Game 🎮

A simple text-based Hangman Game built using Python.

The player has to guess the hidden word one letter at a time before running out of chances.

📌 Features

Random word selection

Limited incorrect guesses

Interactive console gameplay

Win/Lose conditions

Beginner-friendly project

🛠️ Technologies Used

Python

Random Module

Loops & Conditional Statements

Functions

🎯 Game Rules

You need to guess the hidden word letter by letter.

You have only 6 incorrect attempts.

If you guess the word correctly before attempts finish, you win.
